== Description ==

Accept payment on your GiveWP plugin through multiple payment channels via lekirpay payment gateway.

Signup lekirpay merchant account [here](https://app.lekirpay.com/) for live or [here](https://sandbox.lekirpay.com/) for sandbox.


== Frequently Asked Questions ==

= Where can i get Client ID, lekirKey and lekir-signature Key ? =

* Signin lekirpay merchant account

* Make sure account is verified

* Go to the top right of the menu
![first](https://bxcess.sgp1.digitaloceanspaces.com/1.JPG)

* Click the store icon, next click profile button
![second](https://bxcess.sgp1.digitaloceanspaces.com/2.JPG)

* Navigate to API tab
![third](https://bxcess.sgp1.digitaloceanspaces.com/3.JPG)
